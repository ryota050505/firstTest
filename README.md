# firstTest

猫が頼んでくるのに何故か猫を操作して穴を掘らせるゲームです。
binフォルダ内のゲームボーイアドバンスのアイコンがmbファイルを動かせるソフトです。
コンパイル時にはバッチファイルをダブルクリックして活用してください。


また、このGBAアプリでは

GBA上のキー	->	対応するキーボードのキー
	A	->		z
	B	->		x
       START	->	       Enter
      SELECT	->	   Back Space

が対応しています。移動の際はそのまま矢印キーを使用してください。
色によってゴールが違うので、お暇な方は全色ゴールしてみてください。
